# Apex Legends Tracker
Track somebody's stats on Apex Legends.

## Forks welcome
Want to add more stuff to this project? Create a fork, then create a pull request!

## Installation

`npm install`
<br>
<br>
`cd client`
<br>
`npm install`
<br>
##### Back in the main folder
`npm run dev`
